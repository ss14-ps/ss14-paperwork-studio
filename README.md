# SS14 Paperwork Studio

SS14 Paperwork Studio сделан на HTML, CSS и JavaScript.

Это браузерный инструмент для работы с документами в Space Station 14. В нём можно заполнять и редактировать формы, сохранять черновики, создавать профили персонажей, назначать участников форм и сразу смотреть, как документ будет выглядеть с SS14-разметкой.

Репозиторий: https://github.com/ss14-ps/ss14-paperwork-studio

Авторы и благодарности указаны в `CREDITS.md`.

Готовый документ можно скопировать и вставить в игру.

Все данные хранятся локально в браузере через `localStorage`. Проект можно запустить обычным открытием файлов или разместить на GitHub Pages.

## Последние изменения

- Немного переделан рабочий экран.
- Поля стали проще, лишние статусы убраны.
- Для разных форм добавлены роли участников.
- Карточки участников теперь обновляются при ручном вводе.
- Добавлена тема Retro 98 с элементами старого интерфейса и изображениями из вики SS14.
- Исправлен предпросмотр SS14-тегов внутри полей.
- Исправлены формы, где поля участников работали неправильно или повторялись.

## Файлы

- `index.html` - основное приложение.
- `data/forms.json` - список форм.
- `data/forms-data.js` - тот же список форм в виде preload-скрипта для запуска без сервера.
- `data/departments.json`
- `data/jobs.json`
- `data/clauses.json`
- `data/field-presets.json`

Остальные JSON-файлы содержат данные, которые используются в формах.
